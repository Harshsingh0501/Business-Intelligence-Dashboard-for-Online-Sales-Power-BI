#Business Intelligence Dashboard for Online Sales (Power BI

This project is an interactive Power BI dashboard built to analyze online sales data across products, regions, and time periods.

## Features
- Sales, revenue, and profit analysis
- Dynamic filters and slicers
- Drill-down by product, category, and region
- KPI tracking using DAX measures

## Tools & Technologies
- Power BI Desktop
- DAX
- Data Modeling (Relationships, Joins)
- Data Visualization

## Files
- `Online_Sales_Dashboard.pbix` → Main Power BI file  
- `Dashboard_Preview.pdf` → Dashboard view  
- `Screenshots/` → Visual previews

## How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use slicers and filters to explore the data
